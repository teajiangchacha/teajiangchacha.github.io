# 《青苹果事件》

点击这里直达 -->《青苹果事件》

伪姐弟 / 白切黑 / 疯疯的

简介：

（1）

在梁初楹跟暗恋对象确定关系的那一天，她弟弟梁聿自杀了。

她在梁聿的棺材前流了几滴眼泪，尽管她和这个假弟弟的感情并不太好。

如果再给她一次重来的机会，梁初楹绝对不会再欺负他，给梁聿一个完美童年，让他开开心心活到二十岁，然后再死也不迟。

离谱的是，立下誓言的第二天，她真的回到了十七岁那年，更离谱的是…

她发现梁聿对她好像早就有了别的心思。


（2）

六岁时，梁初楹的父母大吵一架，第二天父亲领回来一个小孩，母亲愤愤离家出走，她家庭破裂。

她坚信那是父亲外遇的孩子，从不曾给他好脸色。

梁初楹往梁聿的床上塞过老鼠玩具，他下眼眶泛了红，蹲在她床边，小心翼翼问她能不能一起睡，他只需要小小的一角就好，不会占太多地方。

她故意往梁聿的水杯里加药，他当天夜里发了烧，一边打针一边握住她的手指，皮肤透着红，哀求她别走。

她毁掉了梁聿送给自己的生日礼物，他捧着断掉的手链，漆黑的眼瞳静若寒潭，却发着可怜的嗓音：“好可惜，本来是一对的……”

重回十七岁，梁初楹顾及这人最后的凄惨结局，决定跟他好好相处，于是偷偷往他书桌上放了一个青苹果，宣布止战。

结果当晚，她的房门被敲响，梁聿捏着那个青苹果笑意盈盈地看着她，轻声念：“这是什么意思呢，姐姐？”

“为什么突然不恨我了？”

“是找到让你更讨厌的人，所以注意力都跑到他那里去了吗？”

“不行哦。” 他眉眼更弯，笑意却不达眼底，“要好好关注我啊，梁初楹。”

大事不妙……

按照一般簧文的发展，她可能要被 OOXX 了。

# 《山城巷少年》

点击这里直达 -->《山城巷少年》

一场奇幻的重逢，颠倒的黑白，倒影里的真相

简介：

因为人生太操蛋，陈淮决定去死。

廉租房旁边有一家拿大喇叭嚷嚷清仓甩卖的服装店，他买下了一件五十块的棉服，决定体面地去死。

在他拉开家里的衣柜准备拿出早就准备好的麻绳时，从衣柜里掉出来一团蓝色的人形物体。

他淡然拎着麻绳，问：“你谁？”

蓝色物体：“我是一具忧郁的蓝色尸体。”

陈淮：“哦，不认识，离开我家，谢谢合作。”

如果不是他发神经了，就是神经病潜入他家的衣柜了。

后来蓝色物体渐渐长成一个人，她说她叫秦瑶，陈淮对这个名字一点印象都没有。

老旧窗台上的红色不倒翁被秦瑶戳得左摇右晃，窗户外面是连成一片的铁线莲，她亲手串的风铃挂在窗沿叮铃铃响。

秦瑶茫然地看着窗外，说：“我总是记得要去山城巷里的一个寺庙里祷告，要给马蹄街 136 号的住户写一封信。”

夜风撩起她的头发，少女声音怅惘：

“可是我怎么都想不起来，那个山城巷的少年到底是谁。”

【你之于我，是旧事重提。

# 《欲壑》

点击这里直达 -->《欲壑【先婚后爱】》

假清纯落难千金 × 性冷（yu）淡（qiang）矜贵公子


简介：

（1）

海城降下一场十年内程度最大的暴雨，这件事被当地新闻频道大肆报道。

在这样的极端天气下，严家别墅的花园外站着一个人，白色外套被雨水打湿，落入泥水中，乌发雪肤的人木着一张脸，毫无昔日上流名媛的娇贵模样。

家里的管家撑着伞请她进屋谈话，温栀筝终于见到了严老爷子。

开口第一句话，她镇定地说：“我手上有我父母为你们严氏做事的全部证据，一切的资金往来和流水调度都在我这里。”

严老爷子正眼看她：“你想要什么？”

“我要你儿子娶我。”

严老爷子：“…… 我儿子是性冷淡，你不介意就好。”

温栀筝：“……”

你不如直接说他阳。萎。


（2）

严氏作为国内数一数二的金融龙头企业，前董事长五十多岁老来得子，生了个小皇帝，生来就身负黄金万两，只是他深入简出，极少在众人前露面。

而严淮之第一次登上报纸，是因为与那位落魄后的温家独女喜结连理。

照片上的男人西装革履，丹凤眼上挑，薄唇拉得平直，眉心不耐皱起。

女人乌发绾起，嘴角毫无笑意，眉眼清冷疏离，对这种喜事似乎毫无兴趣。

网友锐评：这是哪家名门在办丧事。

婚后几年，温栀筝劈腿绯闻频出，原因竟是丈夫不举所以外出偷腥。

她一边看新闻一边骂骂咧咧往脖子上涂遮瑕，床上那人好整以暇地支着头笑眼睨她。

“麻烦太太帮我澄清谣言了。”

温栀筝至今想不通，曾经那个禁欲到一碰女人就呕吐的人，怎么就变成了霸总文里 “三天三夜” 的变态。

先收藏喜欢的类型更有可能先开哦～
